# Development setup

### Backend 

  - Linguagem e Framework: NodeJS e ExpressJS.

### Frontend

  - Linguagem: ReactJS.
  
### Mobile

  - Linguagem: React Native

### Tests
  - Teste de rotas no backend: Insomnia.
  - Testes integrados: Jest e Supertest.
